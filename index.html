Git命令列表

创建新分支：git branch branchName

切换到新分支：git checkout branchName

git config

git diff -cacheed

git diff head

git init

git status -s

git reset

M 版本库和中间状态有差异

M 工作区和当前文件有差异

http://ndpsoftware.com/git-cheatsheet.html#loc=remote_repo;

git stash

git stash pop

git reset head 重置暂存区

git show 用法

git log –pretty=raw –graph 872b140

git reflog show master

git reflog -1 最近一条

git rest –soft 暂存区工作区不变

1、已经修改某一文件， 但是没有执行 git add，如果撤销
git checkout a.txt

2、已经修改某一个文件，但是已经执行了 git add ，如何撤销
git rest a.txt

3、已经修改了某几个文件，但是没有执行 git add，想撤销到某个版本，但是暂存区、工作区不想撤销
git rest –soft commitId

4、果已经修改了某几个文件，但是没有执行 git add，也提交了暂存区，想撤销到某个commit
git rest –hard commitId

5、如何查看HEAD指向
cat ../.git/HEAD

8、显示简要的增改行数统计,每次提交文件的变更统计，-n 同上，前n条。
git log –stat -n

9、git log 打印如何美化
$ git log –pretty=format:”%h -%an,%ar : %s” -3

10、git stat -s 命令会出现两个MM，分别是什么意思。
第一个M 版本库和中间状态有差异
第二个M 工作区和当前文件有差异
